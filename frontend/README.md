# frontend

## Project setup
```
npm install -g @vue/cli
vue create frontend
cd frontend
npm i
npm i --save-prod axios vuex core-js
npm i --save-dev vue-loader-v16
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
